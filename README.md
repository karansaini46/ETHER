# ETHER

ETHER is a browser-based repository graph explorer with a React client and an
Express API server.

## Development

Create a local environment file and start both development servers:

```bash
cp .env.example .env
npm run dev
```

The Vite client runs on `http://localhost:5173` and the Express server runs on
`http://localhost:3001`.

## Commands

```bash
npm run lint
npm run typecheck
npm run build
```
